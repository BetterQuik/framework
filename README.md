# BetterQUIK

BetterQUIK - это открытый Lua-фреймворк для трейдоров. С его помощью вы сможете быстро
создавать надёжные расширения для торгового терминала QUIK, такие как:

* Торговые Роботы
* Индикаторы
* Алго-заявки
* Советники
* Авто-следование
* Автоматизация
